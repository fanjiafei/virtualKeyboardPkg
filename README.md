# virtualKeyboardPkg
虚拟键盘&amp;测试用例

增加了虚拟键盘的代码和测试程序

virtualkeyboard
虚拟键盘的代码来自于Git
https://github.com/githubuser0xFFFF/QtFreeVirtualKeyboard

qmlappdemo

virtualKeyboardTest
是我为了在嵌入式环境里测试键盘库尔创建的一个简单测试demo


实际测试下来，项目可以在Ubuntu、windows、基于imax 6环境的嵌入式设备中运行。
